Metallideta
reaktsioonid

See peatükk käsitleb reaktsioone kus ei esine metalle lihtainetena.

⎯Reaktsioonid kus üks saadustest on sool⎯

Aluste reageerimine hapetega:
Alus + hape → sool + vesi (H₂O)
3 NaOH + H₃PO₃ → Na₃PO₄ + 3 H₂O
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])

Aluste reageerimine happeliste oksiididega:
Alus + happeline oksiid → sool + vesi (H₂O)
Ca(OH)₂ + CO₂ → CaCO₃ + H₂O
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])

Hapete reageerimine aluseliste oksiididega:
Aluseline oksiid + hape → sool + vesi (H₂O)
ZnO + H₂SO₄ → ZnSO₄ + H₂O
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])

Happeliste oksiidide reageerimine aluseliste oksiididega:
Aluseline oksiid + happeline oksiid → sool
CaO + CO₂ → CaCO₃
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])


⎯Reaktsioonid veega⎯
Aluseliste oksiidide reageerimine veega:
Aluseline oksiid + vesi → alus
Reaktsioon saab toimuda ainult IA ja IIA rühma metallide oksiididega.
Na₂O + H₂O → 2 NaOH
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])

Happeliste oksiidide reageerimine veega:
Happeline oksiid + vesi → hape
SiO₂ ei reageeri.
CO₂ + H₂O → H₂CO₃
(Indeksite muutus on tingitud oksüdatsiooniastmest, vaata [Tuvastamine])
